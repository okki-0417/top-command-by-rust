# top-command-by-rust

Rust で実装した `top` コマンド風 TUI アプリケーション

## スクリーンショット

```
rtop | Sort: CPU | q:終了 c:CPU順 m:MEM順

    PID NAME                           CPU                    MEM
--------------------------------------------------------------------------------
   1234 chrome                         [████████░░░░░░░░░░░░]  40.0% [██████░░░░░░░░░░░░░░]  300M
   5678 ruby                           [████░░░░░░░░░░░░░░░░]  20.0% [████░░░░░░░░░░░░░░░░]  200M
```

## 機能

- プロセス一覧のリアルタイム表示（1秒ごと更新）
- CPU使用率・メモリ使用量をバーグラフで可視化
- メモリ使用量に応じた色分け（緑/黄/赤）
- ソート切り替え（CPU順/メモリ順）
- キーボード操作

## 操作方法

| キー | 動作 |
|------|------|
| `q` | 終了 |
| `c` | CPU順でソート |
| `m` | メモリ順でソート |

## インストール・実行

```bash
# リポジトリをクローン
git clone https://github.com/mumumu/top-command-by-rust.git
cd top-command-by-rust

# インストール（どこからでも rtop で実行可能に）
cargo install --path .

# 実行
rtop
```

または開発時：

```bash
cargo run
```

## 使用クレート

| クレート | 用途 |
|----------|------|
| `sysinfo` | プロセス情報の取得 |
| `crossterm` | ターミナル操作・キー入力 |

## プロジェクト構成

```
src/
├── main.rs      # エントリーポイント、メインループ
├── process.rs   # プロセス情報取得・ソート
└── ui.rs        # 画面表示・バーグラフ描画
```

## 動作環境

- Rust 2021 Edition
- macOS / Linux
